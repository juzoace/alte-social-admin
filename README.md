# Alte admin dashboard

## Description
Admin dashboard for Alte social

## Installation

```bash
$ git clone https://github.com/AlteSocial/alte-admin-dashboard.git
```

rename the `.env.example` to `.env`

```bash
$ npm install
```

## Running the server

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
`
